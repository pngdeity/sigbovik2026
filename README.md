# Flattening the Discrepancy: Gravity-Consistent Model for a Disk-Shaped Earth

Amazing flat Earth paper submitted to [SIGBOVIK 2026](https://sigbovik.org).

## Building the Paper

Install [Typst CLI](https://github.com/typst/typst)

    typst compile main.typ
    
## Running Reconstructions

    pip install -r code/requirements.txt
    python code/slab.py
